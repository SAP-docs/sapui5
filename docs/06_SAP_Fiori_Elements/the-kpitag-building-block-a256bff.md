# The `KPITag` Building Block

The `KPITag` building block provides an easy way to display a simple key performance indicator \(KPI\) in your application.

> ### Sample Code:  
> The `KPITag` building block
> 
> ```
> 
> <macros:KPITag
>   id="mySuccessKPI"
>   number="{TotalPrice}"
>   unit="{CurrencyCode_code}"
>   status="Success"
> />
> ```

For more information and live examples, see the SAP Fiori development portal at [Building Blocks - KPI Tag - Overview](https://ui5.sap.com/test-resources/sap/fe/core/fpmExplorer/index.html#/buildingBlocks/kpiTag/kpiTagDefault).
