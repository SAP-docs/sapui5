# Creating Key Performance Indicators

You can add key performance indicators \(KPI\) to SAP Fiori elements-based applications.

> ### Note:  
> For information about SAP Fiori elements for OData V4, see [Creating Key Performance Indicators](creating-key-performance-indicators-8874399.md).

The key performance indicator \(KPI\) tag is an abbreviated and clickable title with a KPI value.

In SAP Fiori elements, you can create KPI tags using annotations in an analytical service. For more information, see [Creating Key Performance Indicator Tags](creating-key-performance-indicator-tags-101b199.md).
