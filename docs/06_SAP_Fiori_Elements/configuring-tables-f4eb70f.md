# Configuring Tables

You can use the annotations and entries in the `manifest.json` file to control various aspects of tables in SAP Fiori elements for OData V4.



You can use the `UI.LineItem` annotation to configure a table. It can also be packaged within the `UI.PresentationVariant` annotation to control other aspects, such as sorting. For more information about which `UI.PresentationVariant` is picked, see [Configuring Default Settings \(Visualizations, Sort Order, Filter Values\)](configuring-default-settings-visualizations-sort-order-filter-values-49a6ba5.md).

> For information about SAP Fiori elements for OData V2, see [Configuring Tables](configuring-tables-3695931.md).
