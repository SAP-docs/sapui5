# Configuring the AI Notice in a Column Header

You must use the AI notice to show that the content in the table column contains content generated by AI in SAP Fiori elements for OData V4.

The AI notice allows users to make an informed decision regarding the handling of AI-generated data.

> ### Note:  
> You must also provide documentation to users that explains your AI features and the process behind them to ensure transparency.

The AI notice is displayed as an icon in the column header. Clicking the icon opens a popover with configurable content. If no custom content is provided, the following default message is displayed: *This content was partially or fully generated by artificial intelligence \(AI\) technologies. The AI-generated content may contain inaccuracies due to using multiple information sources. Verify results before use.*

You configure the AI notice in a column header in the following ways:

-   As the default text in the `manifest.json` file
-   As plain text in the `manifest.json` file
-   As rich text using a custom fragment in the `manifest.json` file
-   As the default message in the `Table` building block using a column override
-   As custom content in the `Table` building block using a column override



## Using the `manifest.json` File

You configure the AI notice in a column header using the `manifest.json` file in the following ways:

-   As the default text
-   As plain text
-   As rich text using a custom fragment



### As Plain Text

To display the default AI notice, add the `aiNotice` property under your column in the `manifest.json` file and leave it empty, as shown in the following sample code:

> ### Sample Code:  
> `manifest.json`
> 
> ```
> "sap.ui5": {
>     "routing": {
>         "targets": {
>             "ListReportA": {
>                 "options": {
>                     "settings": {
>                         "controlConfiguration": {
>                             "@com.sap.vocabularies.UI.v1.LineItem": {
>                                 "columns": {
>                                     "DataField::PropertyA": {
>                                         "aiNotice": {}
>                                     }
>                                 }
>                             }
>                         }
>                     }
>                 }
>             }
>         }
>     }
> }
> ```



### As Custom Text

  
  
**AI Notice in a Column Header Configured As Plain Text**

![The AI Notice is displayed with plain text in a column header with the title: Created with AI and the text: This is an AI notice for the Simple Property.](images/AI_Notice_in_a_Column_Header_e4a1e4b.png "AI Notice in a Column Header Configured As Plain Text")

To configure the AI notice as plain text, add the `aiNotice` property under your column in the `manifest.json` file and specify your `contentText`, as shown in the following sample code:

> ### Sample Code:  
> `manifest.json`
> 
> ```
> "sap.ui5": {
>     "routing": {
>         "targets": {
>             "ListReportA": {
>                 "options": {
>                     "settings": {
>                         "controlConfiguration": {
>                             "@com.sap.vocabularies.UI.v1.LineItem": {
>                                 "columns": {
>                                     "DataField::PropertyA": {
>                                         "aiNotice": {
>                                             "contentText": "This column contains AI-generated content."
>                                         }
>                                     }
>                                 }
>                             }
>                         }
>                     }
>                 }
>             }
>         }
>     }
> }
> ```

To configure plain text for custom columns, add the `aiNotice` property under your custom column in the `manifest.json` file and specify your `contentText`, as shown in the following sample code:

> ### Sample Code:  
> ```
> "customColumn": {
>     "header": "Custom Column Header",
>     "template": "AppA.fragments.CustomColumnTemplate",
>     "aiNotice": {
>         "contentText": "This column contains AI-generated recommendations."
>     }
> }
> ```

To configure the AI notice as rich text, add the `aiNotice` property under your column in the `manifest.json` file and specify your custom fragment as the `contentFragmentName`, as shown in the following sample code:

> ### Sample Code:  
> `manifest.json`
> 
> ```
> "DataField::PropertyA": {
>     "aiNotice": {
>         "contentFragmentName": "appA.fragments.AiNoticeContentA"
>     }
> }
> ```

> ### Note:  
> If you provide both the `contentText` and `contentFragmentName` properties, the `contentFragmentName` property is used.



## Using the `Table` Building Block

You configure the AI notice in a column header using the `Table` building block in the following ways:

-   As the default message using a column override
-   As custom content using a column override



### As the Default Message

To configure the AI notice as the default message in the `Table` building block, use a column override with the `aiNotice` property as shown in the following sample code:

> ### Sample Code:  
> Default AI Notice Message in the `Table` Building Block
> 
> ```
> <macros:Table metaPath="_myNavProp/@com.sap.vocabularies.UI.v1.LineItem" header="My table header">
>       <macros:columns>
>             <macrosTable:ColumnOverride key="DataField::ChildProperty">
>                   <macrosTable:aiNotice>
>                         <macrosTable:ColumnAINotice />
>                   </macrosTable:aiNotice>
>             </macrosTable:ColumnOverride>
>       </macros:columns>
> </macros:Table>
> ```



### As Custom Content

To configure the AI notice as custom content in the `Table` building block, use a column override with the `ColumnAiNotice` aggregation as shown in the following sample code:

> ### Sample Code:  
> Custom AI Notice Content in the `Table` Building Block
> 
> ```
> <macrosTable:ColumnOverride key="DataField::PropertyA">
>     <macrosTable:aiNotice>
>         <macrosTable:ColumnAINotice>
>             <Text xmlns="sap.m" text="This data was generated by AI. Please verify." />
>         </macrosTable:ColumnAINotice>
>     </macrosTable:aiNotice>
> </macrosTable:ColumnOverride>
> ```
