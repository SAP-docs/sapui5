# SAP Fiori Launchpad Sandbox

Learn how to install and work with the launchpad sandbox.

SAP Fiori launchpad sandbox is a modernized sandbox implementation for local application development and testing, that

-   supports SAPUI5 1.x-legacy-free standards and modern bootstrap mechanisms
-   simplifies local development and automated testing
-   reduces maintenance overhead and technical debt
-   provides a declarative configuration approach
-   works out-of-the-box with a minimal setup

    > ### Note:  
    > The sandbox is intended for local development and automated testing only - it is NOT suitable for productive use.
